# Academic_Managing_System
<h3>리눅스용 학사 관리 프로그램</h3>

<hr>

시스템 프로그래밍 전공 수업 학기말 프로젝트로 진행한 과제.</br>
리눅스 전용 프로그램으로 학생 정보를 입력, 수정, 삭제할 수 있는 프로그램이다.</br>
C 언어로 작성되었고, 이진 파일을 이용한다.</br>
교수전용과 학생전용 두 가지 버전으로 이용할 수 있고 사용법은 아래와 같다.</br>
